Detailed description

Introduction to the concept of FRP (Functional Reactive Programming), supplemented with a guided tour through an example implementation of Game of Life using the reactive-banana and threepenny-gui libraries. Subsequent discussion of the reflex library and FRP as a possible replacement for monadic IO. See video URL for a detailed summary.
